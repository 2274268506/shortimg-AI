## 🎨 颜色方案

### 短链卡片
- **背景渐变**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **文字颜色**: `#ffffff`
- **按钮背景**: `rgba(255, 255, 255, 0.15)`
- **按钮悬停**: `rgba(255, 255, 255, 0.2)`

### 短链按钮
- **主题色**: Element Plus Primary Blue (#409eff)
- **悬停效果**: 轻微提升和变暗
- **图标**: Link 图标

### 标签颜色
- **有短链**: `type="primary"` (蓝色)
- **无短链**: `type="info"` (灰色)
- **成功提示**: `type="success"` (绿色)

## 💬 用户交互流程

### 复制短链流程
```
用户操作          系统反馈
   │
   ├─ 点击"复制短链"按钮
   │        │
   │        ├─ 调用 copyToClipboard()
   │        │
   │        ├─ 尝试现代 API
   │        │  ├─ 成功 → "短链已复制到剪贴板" ✅
   │        │  └─ 失败 → 降级方案
   │        │
   │        └─ 降级方案（兼容旧浏览器）
   │           ├─ 创建 textarea
   │           ├─ 选中文本
   │           ├─ execCommand('copy')
   │           └─ 成功提示 ✅
   │
   └─ 粘贴使用
```

### 生成二维码流程
```
用户操作          系统反馈
   │
   ├─ 点击"二维码"按钮
   │        │
   │        ├─ 打开对话框
   │        │
   │        ├─ 显示加载动画 ⏳
   │        │
   │        ├─ 动态导入 qrcode 库
   │        │
   │        ├─ 生成二维码
   │        │  ├─ 成功 → 显示二维码 ✅
   │        │  └─ 失败 → 错误提示 ❌
   │        │
   │        └─ 缓存二维码（避免重复生成）
   │
   ├─ 下载二维码
   │        │
   │        └─ 下载 PNG 文件 ✅
   │
   └─ 关闭对话框
```

